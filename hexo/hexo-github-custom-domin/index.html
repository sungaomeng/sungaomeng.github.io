Hexo Github 自定义域名
发表于 2019-05-01| 更新于 2019-05-01|Hexo
字数总计:99|阅读时长: 1 分钟
| 阅读量:

修改Github Settings

Settungs -> GitHub Pages -> Custom domain

输入自定义域名后点击 “Save”

踩坑

hexo deploy后站点404

虽然在Github里设置了自定义域名,但每次hexo deploy后站点就404了
再次去github settings里查看域名配置发现被还原了

将域名保存到./public/CNAME中

保存后再次hexo deploy 就不会被还原了
